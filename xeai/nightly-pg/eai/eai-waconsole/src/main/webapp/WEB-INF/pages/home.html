INTEGRATION MIDDLEWARE MODULES
System State

Active

IMHTTP
Integration Middleware Hyper Text Transport Protocol, Provide all method based on HTTP protocols. Featured with most of HTTP method, include ReST and SOAP/Web-service. Comprehend as Middle-tier application that handles routings, message distribution, mapping/transformation and message processing and happened to be a open service in airportaxpert products. Multi integration type such as: Between Synchronous System Between Asynchronous System Between Synchronous and Asynchronous System Protocols transformation such as HTTP to FTP, SMTP, JMS or into HTTP Protocol method and able supporting any message type such as Xml, Json, Fixlenth, CSV, Base64, include Java serialized object.
System State

Active

IMB
Integration Middleware Broker, Supporting as Internal integration module in airportxpert. Use Java-Proxy, commonly name Enterprise Java Bean as exchanges data media. Established using fix-length text message, this make processing message as light processes. Build to support rapid data exchanges and message distribution capability. Using SPEL as message filters and featured Web-service protocols with predefined service definition.

Active

IMSTUB
Integration Middleware Stubings, A responder module, Commonly use as one of Integration test tools, supporting integration that roles system target that provide respected message and logic. With this the integration system will be able to be happened even tough one of related system is not ready or unavailable. Support both, Synchronous / Asynchronous integration type. Support most of common protocols type and this doesn’t’ tied to one of message type, profiling Customable.

IMSCRIB
Integration Middleware SusCRIBe, Roles as Producer/Consumer or Publisher/Subscriber data. Independent to any system type or common protocols. Real-time data distributor, happened to be direct message from one source and distributed to it’s targeted system. Most of the protocols is a custom/common listener. Message broker queue, JMS, FTP. Support to do exchanges protocols, such as http, mails or store it to database.
